こういう時は自分で直すしかないですね！

## Amino を使う

<BookMark
  title="Amino: Live CSS Editor"
  href="https://chrome.google.com/webstore/detail/amino-live-css-editor/pbcpfbcibpcbfbmddogfhcijfpboeaaf"
/>

Amino は自分でサイト毎にスタイルシートを上書きできる拡張機能。

会社のプロダクトの管理画面もこれで上書きしたりして快適に使ってます。

まずはデベロッパーツールで要素を取得していく。

#video_container という div に video 要素がラップされているようだ。

  width={1426}
  height={722}

/>

コードをみてみると width と height が決め打ちされてますね。
